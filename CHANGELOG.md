# Changelog

All notable changes to **FileGym** will be documented in this file.

## [1.0.0] - 2026-06-29

### Added
- Rebranded application from "Morph" to **FileGym**.
- Integrated smart universal conversion pipeline with magic bytes detection.
- Core converters: PDF to PPTX, Markdown to PDF, Markdown to TXT, and PDF to ZIP image extractor.
- Spreadsheet processor supporting JSON, CSV, and XLSX cross-conversions.
- Custom image resizer (bulk-resize JPG, PNG, and WebP).
- Custom image stitcher (combine photos into a single PDF).
- Built-in ZIP compiler.
- Offline-first configuration: removed all unnecessary network permissions for enhanced user security.
- Security configurations: set `usesCleartextTraffic="false"` in the Manifest.
- Package visibility declarations for external document sharing.
- Launcher icons and premium native splash screen branding.
